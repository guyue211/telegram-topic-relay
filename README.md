# Telegram Topic Relay

[中文文档](./README.zh-CN.md) | [English](./README.en.md)

Telegram Topic Relay is a webhook-based Telegram bot service that routes private chats into forum topics in a management group.

- Private chat -> management group topic
- Reply directly inside topic
- Auto create/recreate topic
- Profile card, tags, ban/unban, history
- Webhook mode, suitable for long-running deployment
